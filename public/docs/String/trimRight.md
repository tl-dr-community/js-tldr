# String.prototype.trimRight()

OPICANIE

## Syntax

```js
str.trimRight()
```

## Usage examples

```js
var str = '   foo  ';

console.log(str.length);  // -> 8

str = str.trimRight();
console.log(str.length); // -> 5
console.log(str);       // -> '   foo'
```
