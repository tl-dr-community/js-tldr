# String.prototype.trimLeft()

OPICANIE

## Syntax

```js
str.trimLeft()
```

## Usage examples

```js
var str = '   foo  ';

console.log(str.length); // -> 8

str = str.trimLeft();
console.log(str.length); // -> 5
console.log(str);        // -> 'foo  '
```
