# String.prototype.fixed()

The `fixed` method creates a <tt> HTML element that causes a string to be displayed in fixed-pitch font.
  
## Syntax

```js
str.fixed()
```

## Usage examples

```js
var worldString = 'Hello, world';
console.log(worldString.fixed()); // -> "<tt>Hello, world</tt>"
```
