# String.prototype.toLowerCase()

The **`toLowerCase`** method returns the `string` converted to lower case.

## Syntax

```js
const lowerString = string.toLowerCase();
```

## Usage examples

```js
'ALPHABET'.toLowerCase(); // -> 'alphabet'
'Alphabet'.toLowerCase(); // -> 'alphabet'
```
