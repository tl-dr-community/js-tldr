# Array.prototype.toSource()

The `toSource` method returns a string representing the source code of the array.

## Syntax

```js
arr.toSource()
```

## Usage examples

```js
var alpha = new Array('a', 'b', 'c');

alpha.toSource(); // -> returns ['a', 'b', 'c']
```
