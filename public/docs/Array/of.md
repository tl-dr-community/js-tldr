# Array.of()

The **`Array.of`** method creates a _new_ `array` from a variable number of `arguments`.

## Syntax

```js
const newArray = Array.of(element0, element1, ..., elementN);
```

## Usage examples

```js
Array.of(1); // -> [1]
Array.of(1, 2, 3); // -> [1, 2, 3]
Array.of(undefined); // -> [undefined]
```
