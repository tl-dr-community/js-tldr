# Math.tanh()

The **`tanh()`** method returns the `hyperbolic tangent` of a `number`.

## Syntax

```js
const hyperbolicTangent = Math.tanh(number);
```

## Usage examples

```js
Math.tanh(0);           // -> 0
Math.tanh(Infinity);    // -> 1
Math.tanh(1);           // -> 0.7615941559557649
```
