# Math.log10()

The **`log10()`** method returns the base 10 `logarithm` of a `number`.

## Syntax

```js
const logarithm = Math.log10(number);
```

## Usage examples

```js
Math.log10(0);  // -> -Infinity
Math.log10(1);  // -> 0
Math.log10(10); // -> 1
```
