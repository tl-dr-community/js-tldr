# Math.hypot()

The **`hypot()`** method returns the `square root` of the sum of squares of its `arguments`.

## Syntax

```js
const squareRoot = Math.hypot(argument1, argument2, ...);
```

## Usage examples

```js
Math.hypot(3, 4);       // -> 5
Math.hypot(3, 4, 5);    // -> 7.0710678118654755
Math.hypot();           // -> 0
```
