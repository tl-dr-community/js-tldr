# Math.atanh()

The **`atanh()`** method returns the the `hyperbolic arctangent` of a `number`.

## Syntax

```js
const hyperbolicArctangent = Math.atanh(number);
```

## Usage examples

```js
Math.atan2(90, 15); // -> 1.4056476493802699
Math.atan2(15, 90); // -> 0.16514867741462683
```
