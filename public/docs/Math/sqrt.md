# Math.sqrt()

The **`sqrt()`** method returns the `square root` of a `number`.

## Syntax

```js
const squareRoot = Math.sqrt(number);
```

## Usage examples

```js
Math.sqrt(9);   // -> 3
Math.sqrt(2);   // -> 1.414213562373095
Math.sqrt(1);   // -> 1
```
