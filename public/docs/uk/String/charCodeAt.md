# String.prototype.charCodeAt()

The **`charCodeAt`** method returns an `integer` between **0** and **65535** representing the UTF-16 code unit at the given `index` of a `string`.

## Syntax

```js
const integer = string.charCodeAt(index);
```

## Usage examples

```js
'ABC'.charCodeAt(0); // -> 65
```
