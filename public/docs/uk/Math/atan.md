# Math.atan()

The **`atan()`** method returns the `arctangent` (in radians) of a `number`.

## Syntax

```js
const arctangent = Math.atan(number);
```

## Usage examples

```js
Math.atan(1); // -> 0.7853981633974483
Math.atan(0); // -> 0
Math.atan(-0); // -> -0
```
