# Math.trunc()

The **`trunc()`** method returns the `integer` part of a `number` by removing any fractional digits.

## Syntax

```js
const integer = Math.trunc(number);
```

## Usage examples

```js
Math.trunc(13.37); // -> 13
Math.trunc(42.84); // -> 42
Math.trunc(0.123); // -> 0
Math.trunc(-0.123); // -> -0
```
