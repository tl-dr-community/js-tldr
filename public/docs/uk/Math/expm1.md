# Math.expm1()

The **`expm1()`** method returns e<sup>x</sup> - 1, where x is the argument, and e the base of the natural logarithms.

## Syntax

```js
const coshine = Math.expm1(angle);
```

## Usage examples

```js
Math.expm1(0); // -> 0
Math.expm1(1); // -> 1.718281828459045
Math.expm1(-1); // -> -0.6321205588285577
```
