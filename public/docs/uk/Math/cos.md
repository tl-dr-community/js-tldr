# Math.cos()

The **`cos()`** method returns the `cosine` of the specified `angle` in radians.

## Syntax

```js
const cosine = Math.cos(angle);
```

## Usage examples

```js
Math.cos(0); // -> 1
Math.cos(1); // -> 0.5403023058681398
Math.cos(Math.PI); // -> -1
```
