# Math.random()

The **`random()`** method returns a floating-point, pseudo-random `number` between 0 (inclusive) and 1 (exclusive).

## Syntax

```js
const number = Math.random();
```

## Usage examples

```js
Math.random(); // -> 0.84844440085509
Math.random(); // -> 0.343399648245843
```
