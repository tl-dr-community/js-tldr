# Math.floor()

The **`floor()`** method returns the largest `integer` less than or equal to a given `number`.

## Syntax

```js
const integer = Math.floor(number);
```

## Usage examples

```js
Math.floor(45); // -> 45
Math.floor(45.05); // -> 45
Math.floor(45.95); // -> 45
Math.floor(-45.05); // -> -46
Math.floor(-45.95); // -> -46
```
