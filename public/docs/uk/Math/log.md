# Math.log()

The **`log()`** method returns the natural `logarithm` of a `number`.

## Syntax

```js
const logarithm = Math.log(number);
```

## Usage examples

```js
Math.log(0); // -> -Infinity
Math.log(1); // -> 0
Math.log(10); // -> 2.302585092994046
```
