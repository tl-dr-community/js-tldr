# Math.abs()

The **`abs()`** method returns the `absolute value` of a `number`.

## Syntax

```js
const absoluteValue = Math.abs(number);
```

## Usage examples

```js
Math.abs('-1'); // -> 1
Math.abs(-2); // -> 2
Math.abs(null); // -> 0
Math.abs([2]); // -> 2
Math.abs([1, 2]); // -> NaN
```
