# Math.log2()

The **`log2()`** method returns the base 2 `logarithm` of a `number`.

## Syntax

```js
const logarithm = Math.log2(number);
```

## Usage examples

```js
Math.log2(0); // -> -Infinity
Math.log2(1); // -> 0
Math.log2(10); // -> 3.321928094887362
```
