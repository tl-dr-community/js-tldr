# Math.acosh()

The **`acosh()`** method returns the `hyperbolic arc-cosine` (in radians) of a `number`.

## Syntax

```js
const hyperbolicArccosine = Math.acosh(number);
```

## Usage examples

```js
Math.acosh(-1); // -> NaN
Math.acosh(0); // -> NaN
Math.acosh(0.5); // -> NaN
Math.acosh(1); // -> 0
```
