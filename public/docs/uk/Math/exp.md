# Math.exp()

The **`exp()`** method returns E<sup>x</sup>, where x is the `argument`, and e is Euler's number.

## Syntax

```js
const coshine = Math.exp(angle);
```

## Usage examples

```js
Math.exp(0); // -> 1
Math.exp(1); // -> 1.5430806348152437
Math.exp(-1); // -> 1.5430806348152437
```
