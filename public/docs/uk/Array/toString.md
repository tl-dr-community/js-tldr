# Array.prototype.toString()

The **`toString`** method returns a `string` representing the specified `array` and its elements.

## Syntax

```js
const string = array.toString();
```

## Usage examples

```js
['Am', 'I', 'Array', 'Or', 'String', '?'].toString(); // -> "Am,I,Array,Or,String,?"
```
