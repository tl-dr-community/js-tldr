# Number.prototype.valueOf()

The **`valueOf`** method returns the wrapped primitive `value` of a Number object.

## Syntax

```js
const value = number.valueOf();
```

## Usage examples

```js
const numberObj = new Number(10);
typeof numberObj; // -> object
numberObj.valueOf(); // -> 10
```
