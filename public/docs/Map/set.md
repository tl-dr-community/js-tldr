# Map.set

the **`set`** method adds or updates an element with a specified key and a value to a `Map` object.

## Syntax

```js
map.set(key, value);
```

## Usage Examples

```js
let myMap = new Map();

// Add new elements to the map
myMap.set('bar', 'foo');
myMap.set(1, 'foobar');

// Update an element in the map
myMap.set('bar', 'baz');
```
