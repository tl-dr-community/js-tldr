# Map.get

the **`get`** method returns a specified element from a `Map` object.

## Syntax

```js
map.get(key))
```

## Usage Examples

```js
let myMap = new Map();
myMap.set('bar', 'foo');

myMap.get('bar'); // Returns "foo"
myMap.get('baz'); // Returns undefined
```
