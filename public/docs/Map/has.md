# Map.prototype.has()

the **`has`** method returns a boolean indicating whether an element with the specified key exists or not.

## Syntax

```js
map.has(key))
```

## Usage Examples

```js
const map = new Map();
map.set('bar', 'foo');

map.has('bar'); // -> true
map.has('baz'); // -> false
```
