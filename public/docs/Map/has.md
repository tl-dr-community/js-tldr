# Map.has

the **`has`** method returns a boolean indicating whether an element with the specified key exists or not.

## Syntax

```js
map.has(key))
```

## Usage Examples

```js
let myMap = new Map();
myMap.set('bar', 'foo');

myMap.has('bar'); // returns true
myMap.has('baz'); // returns false
```
