# Map.size

the **`size`** property accessor property returns the number of elements in a `Map` object.

## Usage Examples

```js
var myMap = new Map();
myMap.set('a', 'alpha');
myMap.set('b', 'beta');
myMap.set('g', 'gamma');

myMap.size; // 3
```
