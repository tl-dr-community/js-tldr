# Map.prototype.size

the **`size`** property accessor property returns the number of elements in a `Map` object.

## Usage Examples

```js
const map = new Map();
map.set('a', 'alpha');
map.set('b', 'beta');
map.set('g', 'gamma');

map.size; // -> 3
```
