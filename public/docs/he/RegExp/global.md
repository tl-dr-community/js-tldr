# RegExp.prototype.global

The **`global`** property indicates whether or not the **"g"** flag is used with the `regular expression`. **`global`** is a read-only property of an individual regular expression instance.

## Usage examples

```js
/foo/g.global; // -> true
/foo/i.global; // -> false
```
