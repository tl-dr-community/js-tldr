# Math.min()

The **`min()`** method returns the `smallest number` of given `numbers`.

## Syntax

```js
const smallestNumber = Math.min(number1, number2, ...);
```

## Usage examples

```js
Math.min(10, 20); // -> 10
Math.min(-10, -20); // -> -20
Math.min(); // -> Infinity
```
