# Math.acos()

The **`acos()`** method returns the `arccosine` (in radians) of a `number`.

## Syntax

```js
const arccosine = Math.acos(number);
```

## Usage examples

```js
Math.acos(-1); // -> 3.141592653589793
Math.acos(0); // -> 1.5707963267948966
Math.acos(0.5); // -> 1.0471975511965979
Math.acos(1); // -> 0
```
