# Math.asin()

The **`asin()`** method returns the `arcsine` (in radians) of a `number`.

## Syntax

```js
const arcsine = Math.asin(number);
```

## Usage examples

```js
Math.asin(-1); // -> -1.5707963267948966
Math.asin(0); // -> 0
Math.asin(0.5); // -> 0.5235987755982989
Math.asin(1); // -> 1.5707963267948966
```
