# Math.ceil()

The **`ceil()`** method rounds a `number` up to the next largest whole number.

## Syntax

```js
const roundedNumber = Math.ceil(number);
```

## Usage examples

```js
Math.ceil(0.95); // -> 1
Math.ceil(4); // -> 4
Math.ceil(7.004); // -> 8
Math.ceil(-0.95); // -> -0
```
