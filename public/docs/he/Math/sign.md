# Math.sign()

The **`sign()`** method returns the `integer` part of a `number` by removing any fractional digits.

## Syntax

```js
const integer = Math.sign(number);
```

## Usage examples

```js
Math.sign(13.37); // -> 13
Math.sign(42.84); // -> 42
Math.sign(0.123); // -> 0
Math.sign(-0.123); // -> -0
```
