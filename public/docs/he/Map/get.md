# Map.prototype.get()

the **`get`** method returns a specified element from a `Map` object.

## Syntax

```js
map.get(key))
```

## Usage Examples

```js
const map = new Map();
map.set('bar', 'foo');

map.get('bar'); // -> "foo"
map.get('baz'); // -> undefined
```
