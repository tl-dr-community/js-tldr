# Number.isInteger()

The **`isInteger`** method determines whether the passed `value` is an **integer**.

> NOTE: returns boolean.

## Syntax

```js
const result = Number.isInteger(value);
```

## Usage examples

```js
Number.isInteger(0.1);
// -> false
Number.isInteger(Math.PI);
// -> false
Number.isInteger([1]);
// -> false
Number.isInteger(5.0);
// -> true
```
