# Number.isFinite()

The **`isFinite`** method determines whether the passed `value` is a **finite number**.

> NOTE: returns boolean.

## Syntax

```js
const result = Number.isFinite(value);
```

## Usage examples

```js
Number.isFinite(5);
// -> true
Number.isFinite(1 / 0);
// -> false
Number.isFinite(10 / 5);
// -> true
```
