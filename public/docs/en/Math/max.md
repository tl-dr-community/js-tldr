# Math.max()

The **`max()`** method returns the `largest number` of given `numbers`.

## Syntax

```js
const largestNumber = Math.max(number1, number2, ...);
```

## Usage examples

```js
Math.max(10, 20); // -> 20
Math.max(-10, -20); // -> -10
Math.max(); // -> -Infinity
```
