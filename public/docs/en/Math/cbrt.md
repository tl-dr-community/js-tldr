# Math.cbrt()

The **`cbrt()`** method returns the `cube root` of a `number`.

## Syntax

```js
const cubeRoot = Math.cbrt(number);
```

## Usage examples

```js
Math.cbrt(2); // -> 1.2599210498948732
Math.cbrt(8); // -> 2
Math.cbrt(27); // -> 3
```
