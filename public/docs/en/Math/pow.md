# Math.pow()

The **`pow()`** method returns the `base`<sup>`exponent`</sup> power.

## Syntax

```js
const power = Math.pow(base, exponent);
```

## Usage examples

```js
Math.pow(7, 2); // -> 49
Math.pow(7, 3); // -> 343
Math.pow(2, 10); // -> 1024
```
