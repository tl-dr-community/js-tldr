# Math.cosh()

The **`cosh()`** method returns the `hyperbolic cosine` of a `number`.

## Syntax

```js
const coshine = Math.cosh(angle);
```

## Usage examples

```js
Math.cosh(0); // -> 1
Math.cosh(1); // -> 1.5430806348152437
Math.cosh(-1); // -> 1.5430806348152437
```
