# Math.round()

The **`round()`** method returns the `value` of a `number` rounded to the nearest integer.

## Syntax

```js
const value = Math.round(number);
```

## Usage examples

```js
Math.round(20.49); // -> 20
Math.round(20.5); // -> 21
Math.round(42); // -> 42
Math.round(-20.5); // -> -20
Math.round(-20.51); // -> -21
```
