# Math.sin()

The **`sin()`** method returns the `sine` of a `number`.

## Syntax

```js
const sine = Math.sin(number);
```

## Usage examples

```js
Math.sin(0); // -> 0
Math.sin(1); // -> 0.8414709848078965
Math.sin(Math.PI / 2); // -> 1
```
