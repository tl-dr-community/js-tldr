# String.prototype.valueOf()

The **`valueOf`** method returns the primitive value of a String object.

## Syntax

```js
str.valueOf();
```

## Usage examples

```js
var x = new String('Hello world');
console.log(x.valueOf()); // -> Displays 'Hello world'
```
