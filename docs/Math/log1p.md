# Math.log1p()

The **`log1p()`** method returns the natural `logarithm` of 1 + `number`.

## Syntax

```js
const logarithm = Math.log1p(number);
```

## Usage examples

```js
Math.log1p(0);  // -> 0
Math.log1p(1);  // -> 0.6931471805599453
Math.log1p(10); // -> 2.3978952727983707
```
