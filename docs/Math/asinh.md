# Math.asinh()

The **`asinh()`** method returns the `hyperbolic arcsine` of a `number`.

## Syntax

```js
const hyperbolicArcsine = Math.asinh(number);
```

## Usage examples

```js
Math.asinh(1);  // -> 0.881373587019543
Math.asinh(0);  // -> 0
```
