# Math.sinh()

The **`sinh()`** method returns the `hyperbolic sine` of a `number`.

## Syntax

```js
const hyperbolicSine = Math.sinh(number);
```

## Usage examples

```js
Math.sinh(0);   // -> 0
Math.sinh(1);   // -> 1.1752011936438014
```
